My Projects

Real-Time Aircraft Tracker

This is a live aircraft tracker built using a Raspberry Pi and an ADS-B receiver which captures broadcast signals from nearby planes. The Raspberry Pi then processes this data, plots the planes on a display and uploads them to a Firebase database.
A Kotlin mobile app then retrieves the data from Firebase and compares the aircraft coordinates to the user's current location. If nearby planes are detected, the app notifies the user in real time.
